# POKE_DATA

1. install packages:
      ```bash
      sudo apt install -y python3 python3-venv python3-pip  libpq-dev
      ```


2. Clone the repository:

   ```bash
   git clone https://github.com/knowitmoa/poke_data.git

   cd poke_data

   ```

  



3. Run in terminal:

   ```bash
   bash setup.sh
   ```
